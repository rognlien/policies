# Drop a Crumb

One-tap location saving for travellers. Remember the places you discover.

## Features

- Save your current location with a single tap
- Lock screen widget for hands-free saving
- Reverse geocoding for automatic street addresses
- Attach photos to saved locations
- Siri and Shortcuts integration
